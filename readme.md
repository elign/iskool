## Features

As per the review from the teacher there would be following features in the app:

- Daily Updates
- Notifications
- Academic Progress

### Daily Updates

3 sub-sections:
- Class work
- Home work
- Language

### Notifications

3 sub-sections:
- Monthly Calendar
- Circular & Notices
- Club Activity Details

### Academic Progress

3 sub-sections:
- Class test reports.
- Exam Results
